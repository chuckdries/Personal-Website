TechBuds Chat

A realtime chat app in Vue

A penny stock trading community outgrew the size limit on their Slack team, so they hired us to replace it. Our back end relies on Mongo, Redis, Node, and Express; our front end uses Vue, Vue-router, and Materialize. Socket.io handles real time communication between the two, and every ounce of javascript we write is transpiled by Babel. I'm in charge of the front end and responsible for the visual and architectural design of most of the SPA.

Unfortunately, this project is closed source, but it's not under NDA. I can't publish the code online, but I can show you if you ask.
